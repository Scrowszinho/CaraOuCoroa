# 🐱‍👤 Sobre

Um aplicativo simples criado com react-native
onde o usuário joga uma moeda, tendo a sorte de 
cair cara ou coroa.<br />

* Programa gera um número aleatório. 

* Programa exibe o resultado em imagem. <br/>
